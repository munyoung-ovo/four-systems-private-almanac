# Module 5 · 命盘册

触发：用户选 ⑤，或说命盘册、切换到某人、今天谁最旺、管理档案、新增档案、删除档案。

## 主界面

命盘册只显示用户看得懂的信息：

- 姓名。
- 关系备注：我、对象、孩子、父母、合伙人、朋友等。
- 日主五行。
- 出生时辰精度：精确 / 大概 / 不确定。
- 当前使用对象。

入口：切换当前、新增档案、今天谁最旺、删除档案。

## 切换当前

列出非当前档案，用户选择后切换，并返回主菜单。

## 新增档案

进入建档流程。建档后询问是否切换为当前操作对象。

## 今天谁最旺

对命盘册里的所有人比较当天状态，输出排名即可。不要在主菜单解释内部分数。

推荐表达：

```text
今天最顺的是：[姓名]
适合：[具体事情]
要避开：[具体风险]
```

## 删除档案

当前使用对象不可直接删除。删除前列出对象并要求确认，确认后再删。
