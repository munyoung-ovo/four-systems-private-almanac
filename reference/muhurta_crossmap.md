| 祈福 / 祭祀 | 宗教仪式 | 天德月德、吉日 |
| 搬家 / 入宅 | 定居落根 | 土、生气日 |

## 八字/干支系统投影规则

| 条件 | 投影 |
|------|------|
| 日支与命主四支六合，且合化五行属用神 | favor |
| 日干五行生助日主，当日天乙贵人到位 | favor |
| 建除为"成"日或"开"日，无冲无刑 | favor（加分） |
| 日支冲本命日支或年支 | avoid |
| 当日三煞/岁煞到位 | avoid |
| 其余 | neutral |

**act 专属加权**：
- 签约：官星（正官/七杀）当日透干 → favor +1
- 开业：食神/伤官当日得地 → favor +1
- 出行：驿马动 → favor +1；建除"破"日 → avoid
- 嫁娶：桃花到，天乙贵人到 → favor；冲日主 → avoid
- 动土：建日 → avoid；除日 → favor（除旧布新）

- 嫁娶：流年/流月夫妻宫有吉星且无化忌 → favor
- 出行：迁移宫吉星聚集 → favor

## Nakshatra/印度系统投影规则

### 吉宿（对应 favor）
| Nakshatra | 适合事项 |
|-----------|---------|
| Rohini | 嫁娶、签约、开业（最吉宿） |
| Hasta | 手工、医疗、旅行 |
| Pushya | 投资、祈福、搬家 |
| Uttara Phalguni | 签约、婚姻 |
| Uttara Ashadha | 开业、长期事业 |
| Uttara Bhadrapada | 稳定性事项 |
| Revati | 旅行、珠宝、艺术 |

### 凶宿（对应 avoid）
| Nakshatra | 原因 |
|-----------|------|
| Bharani | 死亡相关，凶星 |
| Krittika | 锋利切割，不宜嫁娶/签约 |
| Ardra | 混乱破坏，忌一切启动 |
| Ashlesha | 欺骗毒蛇，忌法律金融 |
| Jyeshtha | 纷争嫉妒，忌合作 |
| Mula | 根基破坏，忌动土/搬家 |
| Shatabhisha | 秘密孤立，可用于医疗 |

### 中性宿（neutral）
其余 Nakshatras 为中性，视具体 act 微调。

## 西方占星系统投影规则

| 条件 | 投影 |
|------|------|
| 木星过境与本命太阳/月亮/上升形成三合/六合 | favor |
| 金星过境与本命金星/木星合相或三合 | favor（财富/感情类） |
| 月亮当日与本命行星形成正向相位（三合/六合） | favor |
| 土星/火星与本命太阳/月亮形成四分/对分 | avoid |
| 日食/月食当日（新月/满月极近本命行星） | avoid（不宜启动） |
| 水星逆行期间 | avoid（签约/合同类） |
| 其余 | neutral |

**act 专属加权**：
- 签约/合同：水星顺行且与本命水星无压力相位 → favor
- 嫁娶：金星相位吉 → favor
- 投资：木星过境第二宫（财宫） → favor
- 出行：木星过境第九宫（远途） → favor

## 投影合并规则（供 resonance.py 使用）

1. 每系统独立输出 `favor / neutral / avoid`
2. `resonance_strength = 投 favor 的系统数`（计数，不取平均）
3. 仅 **4/4 favor** 才标 `is_destined_moment=true`
4. 出现 avoid 但其他系统 favor → 按 resonance_protocol.md 归类冲突
