# 紫微斗数解读风格与星曜含义

## 解读风格原则

1. **结构优先**：先说"宫位 + 星曜组合"的核心含义，再说倾向与建议
2. **不断言宿命**：「天相在官禄」= "事业有官贵贵人助力的倾向"，不说"你命中注定当官"
3. **阴阳互转**：每颗星都有光明面与阴影面，解读时两面都提，再给出激活光明面的建议
4. **流年为主**：深算时以流年/大限为重心，命盘为背景——"此刻你在走哪段路"比"你是什么人"更有用

## 十二宫位速查

| 宫位 | 主题 | 解读重点 |
|------|------|---------|
| 命宫 | 本质、气质、命运主轴 | 最重要，决定整体格局 |
| 兄弟宫 | 兄弟/朋友/同事关系 | 人际支持网络 |
| 夫妻宫 | 婚姻/伴侣/合作 | 感情与合作模式 |
| 子女宫 | 子女/创作/下属 | 传承与创造 |
| 财帛宫 | 金钱观/理财/财运 | 赚钱方式与财富积累 |
| 疾厄宫 | 健康/身体/逆境 | 体质与处理危机的方式 |
| 迁移宫 | 出行/外地/外缘 | 异地运势与外部环境 |
| 奴仆宫 | 下属/服务/合伙 | 人际下游关系 |
| 官禄宫 | 事业/工作/名誉 | 职业方向与成就 |
| 田宅宫 | 房产/家庭/根基 | 物质基础与家庭关系 |
| 福德宫 | 内心/精神/享乐 | 精神世界与幸福感来源 |
| 父母宫 | 父母/上司/长辈 | 长辈关系与早年环境 |

## 四化解读（化禄/化权/化科/化忌）

| 四化 | 含义 | 落入吉宫 | 落入凶宫 |
|------|------|---------|---------|
| 化禄 | 财气流入、顺遂 | 好运聚焦于此宫 | 财散于此，需留意 |
| 化权 | 权力欲、执行力 | 此宫事项我有主导权 | 强势冲突 |
| 化科 | 名誉、考试、文书 | 此宫得名声贵人 | — |
| 化忌 | 阻碍、纠结、暗损 | 注意此宫事项的消耗 | 重点警惕区 |

**解读原则**：化忌不等于凶，而是"需要特别留心、主动管理的领域"。

## 大限/流年解读框架

```
当前大限（10年周期）：[宫位] × [主星] 
→ 这10年的主题是什么？哪个宫位主导？

流年（当年）：流年命宫落入 [宫位]
→ 今年着力点在哪？

流月：精确到月，用于短期决策

解读顺序：大限背景 → 流年主题 → 流月节点
```
