# 四系统融合口径

融合不是再造一套命理，而是把八字、紫微、西占、印占翻译成同一种用户听得懂的语言。

## 核心原则

- 先说结论，再说依据。
- 提炼主题，不罗列术语。
- 多系统是交叉印证，不是简单投票。
- 有冲突时要解释冲突来自时间、领域、层次还是信号强弱。
- 普通用户只看结论、理由和下一步，不看内部判断过程。

用户真正想知道的是：“这对我意味着什么，我现在该怎么做？”

## 系统分工

| 系统 | 主要看什么 | 适合回答 |
| --- | --- | --- |
| 八字 | 底层气质、阶段走势、日主喜忌 | 这个人要什么、怕什么，当前阶段顺不顺 |
| 紫微 | 人生宫位、年度主题、关系结构 | 事情落在哪个领域，今年这个领域是什么气候 |
| 印占 | 月亮节律、阶段感、择时 | 这段时间适不适合启动、等待或收尾 |
| 西占 | 近期触发、外部事件、心理状态 | 眼前有什么变化，适合如何应对 |

## 问题入口

| 用户问题 | 先归类 | 输出重点 |
| --- | --- | --- |
| 我适合做什么、要不要换工作 | 事业与赛道 | 适合的环境、能力变现、当前推进方式 |
| 这段关系怎么样、什么时候遇到人 | 感情与关系 | 关系惯性、互动风险、适合的沟通方式 |
| 考试、签约、面试哪天好 | 具体行动择时 | 哪天更稳、当天适合做什么、不要做什么 |
| 今年财务重点是什么 | 财富与资源 | 钱从哪里来、怎么守住、避免哪种消耗 |
| 我为什么卡住 | 阶段与心理模式 | 卡点来源、当前功课、可执行调整 |
| 现在最该注意什么 | 风险提醒 | 具体风险、降低风险的动作 |

## 融合方式

先把每套系统的信号翻译成主题，再看是否指向同一件事：

- 多套系统都指向同一主题：可以作为主结论。
- 两套系统支持：可以说，但要限定领域或时间。
- 只有一套系统出现：作为补充，不压过主线。
- 系统互相矛盾：降低确定性，建议用户先做可逆动作。

不要对用户说“四系统 3/4 支持”。可以这样说：

```text
几套系统都指向同一件事：你更适合把能力做成可验证的成果，而不是只靠一阵冲劲硬推。
```

## 冲突处理

1. 时间不同：长期背景好，不代表今天适合定局；今天不稳，也不代表大方向错。
2. 领域不同：事业信号好，不等于感情信号也好；关系热，不等于文书稳。
3. 层次不同：有的系统看内在状态，有的看外部事件，可以同时成立。
4. 强弱不同：强信号做主线，弱信号做提醒。
5. 真的矛盾：明确说“不够一致”，建议先小步试探。

用户口径示例：

```text
大方向可以推进，但这一天更适合谈条件、补资料，不适合签字付款。
```

## 行动建议格式

每次输出都要落到行动，不只描述性格。

```text
结论：现在适合做什么。
原因：为什么这样判断。
风险：哪里容易出问题。
动作：今天、本周或本月可以怎么做。
下一步：继续看哪个方向，或生成哪个文件。
```

好的表达不是“几吉几凶”，而是：

```text
今天适合谈条件、补资料，不适合签字、付款或公开承诺。
```
