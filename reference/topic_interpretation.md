# 专题解读协议

本文件规定不同问题应覆盖哪些判断层，防止用一套性格模板回答所有问题。只读取当前专题对应部分。

## 整体命盘与核验

整体命盘先提炼 3-5 条贯穿多个领域的核心机制，再分别说明优势、代价和当前阶段。不要把四个系统拆成四份报告。

需要核验时选择三条不同来源：一条行动或决策惯性、一条关系或情绪机制、一条资源/事业/压力模式。判断必须具体到可观察行为，并允许用户直接否定。禁止使用“外表坚强内心敏感”“有时外向有时内向”等普遍适用句。

用户反馈后的处理顺序：核对原始资料 → 核对结论层次与时间范围 → 检查是否存在系统冲突或降级 → 必要时承认当前解释不足。反馈不能反向生成盘面事实。

## 感情

依次判断：

1. 亲密需求：安全感来自确认、空间、行动、稳定还是精神交流。
2. 表达与接收：如何表达喜欢，怎样的表达才能真正接住。
3. 吸引模式：容易被哪类气质或互动吸引，吸引是否有长期承载力。
4. 冲突循环：压力下是追问、退开、控制、讨好还是冷处理，双方如何相互触发。
5. 关系边界：金钱、时间、承诺、家庭和个人空间中哪一项最需说清。
6. 当前阶段：只在流运证据完整时说明未来节奏，不预测必然出现某个人、结婚或分手。

## 事业

依次判断：

1. 工作结构：适合高自主、明确规则、专业深耕、项目变化、前台表达还是后台整合。
2. 能力变现：最容易形成价值的能力是什么，靠专业、管理、销售、创作、资源还是执行落地。
3. 决策与协作：速度、风险偏好、权责需求，以及在团队中容易承担的角色。
4. 卡点与代价：优势过度使用会怎样，当前环境是否放大这个问题。
5. 当前阶段：扩张、积累、转换或收缩；区分“有能力”与“现在适合动”。
6. 具体选择：换岗、创业、合作或副业分别说明成立条件、主要风险和最小可逆行动。

不能仅根据一个五行、一个宫位或一颗星给出具体职业名称。职业建议应先给工作性质和环境，再在用户现实条件允许时举例。

## 财运

依次判断：收入形成方式、资源调度能力、守财习惯、风险偏好、合作分配和当前现金流节奏。分别回答“能不能创造收入”“能不能稳定留下”“当前是否适合扩大风险”，三者不能混为一谈。

财星、财富宫或有利行运只说明相关主题被激活，不自动等于盈利。投资、借贷和重大支出必须结合现实财务信息，命盘不得代替专业判断或承诺收益。

## 贵人

依次判断：贵人更可能是哪类角色、通过什么场景出现、能够提供什么资源、用户为何容易错过，以及怎样回应帮助。贵人可以是提供信息、机会、背书、执行协助或情绪稳定的人，不统一解释成有权势者。

没有时间证据时只谈长期渠道，不编造方位、月份和出现次数。单个吉星不能直接推断近期必有人相助。

## 健康与状态

只回答压力模式、精力波动、作息、运动、饮食节律和恢复方式。把本命倾向、当前阶段与当天状态分开，不从五行、宫位或星体推断具体疾病、器官问题和寿命。

出现持续、严重或异常症状时，明确建议按现实医疗渠道处理。命理建议只能作为生活节律的补充。

## 具体事件

先判断问题属于能力、阶段还是日期：

- 能不能：主要看本命能力与现实条件。
- 现在该不该：主要看当前阶段、代价和可逆性。
- 哪天做：主要看明确事项与候选日期。

只有差异确实改变建议时才确认主动/被迫、已决定/评估中、目标日期或现实限制。回答按“直接结论 → 本命影响 → 当前阶段 → 时间触发 → 风险条件 → 下一步”自然展开，不承诺具体结果。
