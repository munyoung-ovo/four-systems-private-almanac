# 黄道吉日 / 私人通勝

把传统择日、八字、紫微、西占、印占放进一个本地 skill。

不是让 AI 随口算命，而是先用确定性脚本排盘，再让 AI 把结果讲人话。

---

## 这是什么

运行在 Codex / Claude Code 上的私人通勝工具。

你给它出生时间、地点和想问的事，它会在本地生成档案、排盘、择日、合盘、日历和图文报告。

适合这些场景：

- 今日运势：今天适合推进什么，哪些事先别硬冲
- 黄道吉日：考试、面试、签约、搬家、表白、出行、开工择日
- 深度命盘：性格、事业、感情、阶段运势的长文解读
- 合盘关系：两个人的吸引、磨合、风险点和相处建议
- 日历导出：把适合行动的日期生成 ICS / HTML，方便查看和分享

---

## 核心功能

1. **建档案**  
   输入姓名、性别、公历出生时间、出生地，经纬度和时区可选。私人档案保存在本地 `profiles/`。

2. **今日算命 / 择日**  
   结合当天通书、个人八字、行运、星曜和跨系统共振，输出适合做什么、不适合做什么。

3. **四系统合参**  
   默认覆盖八字、紫微斗数、西方占星、Vedic astrology。系统会按问题类型和精度排序，不把低精度信号硬说成结论。

4. **深度命盘解读**  
   从确定性排盘结果出发，生成性格、事业、感情、阶段主题等完整解读。

5. **合盘**  
   支持你和另一个人的关系分析，区分吸引力、现实摩擦、长期相处和行动建议。

6. **日历 / 图文输出**  
   可生成 `.ics` 日历和 HTML 图文报告，长结果默认放在 `outputs/`。

---

## 为什么这样做

很多占星项目重在“能画盘”或“能算位置”，比如 Swiss Ephemeris、pyswisseph、PyJHora、AstroChart 这类工具都强调可靠计算。

这个项目的重点是另一半：

- 计算必须本地完成
- 字段必须来自真实引擎返回
- AI 只负责解释、融合、改写和追问
- 资料不上传，个人档案不进仓库
- 择日不是单一系统拍脑袋，而是多系统合参后给人话建议

一句话：排盘交给脚本，表达交给 AI。

## 安装

需要先安装 Python 3.8+。

**方式一：一键安装依赖**

macOS / Linux：

```bash
bash install.sh
```

Windows：

```bat
install.bat
```

或手动运行：

```bash
python install.py
```

安装脚本会安装并检查：

- `lunar_python`
- `iztro-py`
- `pyswisseph`
- `pypdf`

---

## 安装为 skill

我的建议是下载 ZIP 手动安装

点击右上角绿色代码按钮→下载ZIP，解压后将文件夹整体复制到你的项目根目录下。
 

把这个项目安装到 Codex / Claude Code 的 skills 目录：

```bash
python install_skill.py
```

只安装到 Codex：

```bash
python install_skill.py --codex-only
```

只安装到 Claude Code：

```bash
python install_skill.py --claude-only
```

安装后开新对话即可生效。

## 使用

在 Codex / Claude Code 里输入：

```text
载入黄道吉日
```

或直接说：

```text
今天适合签约吗
11月雅思能过吗
帮我找下个月适合面试的日子
给我做深度命盘解读
我和这个人合盘看看
```

首次使用时，如果没有档案，它会引导你输入：

```text
姓名：林xx
性别：女
出生：1998-04-12 09:30
地点：广东广州天河区
关系备注：我
```

主菜单大致长这样：

```text
① 今日算命
② 生成日历/图文
③ 深度命盘解读
④ 合盘
⑤ 命盘册
```

## 数据放在哪里

```text
profiles/   个人出生档案
materials/  用户上传的 PDF / 图片材料
outputs/    生成的长文、日历、HTML 报告
examples/   可公开演示的虚构样例
```

`profiles/`、`materials/`、`outputs/` 默认用于本地私人数据，不建议提交真实出生信息、星盘 PDF 或解读报告。

公开展示用的虚构样例放在 `examples/`：

- `sample_profile_input.json`：示例输入
- `deep_chart_preview_t_val.md`：深度命盘短版
- `action_map_t_val_2026-06.html`：运势行动图
- `relationship_map_20260629_012844.html`：关系说明书

---

## 计算口径

- 八字：立春分年界，12 节气分月界，23:00 换日
- 八字默认使用真太阳时校正
- Vedic：固定使用 Lahiri ayanamsa
- 西占：基于 Swiss Ephemeris / `pyswisseph`
- 紫微：通过 `iztro-py` 生成结构化盘面

如果出生时间接近时辰边界、23:00 换日边界，或缺少准确出生时间，结果会降级标注，不会硬编。

```bash

```

它会检查依赖和金标准样例：

```text
四柱：乙亥 / 己卯 / 壬寅 / 甲辰
月宿：Punarvasu
ayanamsa：Lahiri
```

通过后再使用择日、命盘和合盘功能。

## 项目结构

```text

templates/   HTML 报告模板

```
